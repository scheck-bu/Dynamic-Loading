Keysight 3458A Driver

Introduction:

  This library is for controlling the Keysight 3458A Digital Multimeter and is intended to be used in conjunction with the Digital Multimeter interface.

  Sample usage:

    using DMMInterface;

    // Create new DMM object using GPIB address.
    Assembly DMMAssem = Assembly.LoadFrom(@".\Test Programs\SM1472\AES_ITA_Driver.dll");
    Type DMMType = DMMAssem.GetTypes()[0];
    object[] args = { "22"};
    IDMM dmm = (IDMM)Activator.CreateInstance(DMMType, args);

    // Measure DC voltage
    double reading = dmm.DC.Voltage.measure();

    // Measure two wire resistance
    dmm.Resistance._2W.Range = IDMM.IResistance.I2W.RangeEnum._1M;
    reading = dmm.Resistance._2W.measure();


Library contents:

    ID

      Type: string

      Summary: Gets ID from instrument.

      Usage: string id = dmm.ID;

    Resistance

        Summary: Returns resistance measurement.
